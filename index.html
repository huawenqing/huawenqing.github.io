猫姑娘(七)

发表于 2021-04-26 分类于 小说
本文字数： 3.7k 阅读时长 ≈ 3 分钟

我的男朋友死了，被车压死的。

我迅速和另一只陌生的白猫相识，然后疯狂做爱，在初冬生下一窝猫崽儿。猫崽儿们个个纯正的白色，绵长的绒毛，随我。自从生了孩子，我再也没见那只白猫。我对他的印象仍然新鲜而陌生。我想见见他，也想见见那死去的男朋友。

那个焦急的早晨，我闯进门钻到主人的床上，后来他懂得了我是要生了。我被主人安置在一个铺满破旧棉衣的纸盒子里，然后放到一张堆满了杂物的桌子底下，密不透风，很温暖、很安全。主人每天定时来送吃的，我基本不用出窝，每天所做的就是打着盹给孩子们喂奶，或者用舌头给他们梳理毛发。生孩子的感觉真好，不用在大冷的天奔波着寻找那些冻硬了的剩菜剩饭。

我跋涉一夜，嗅到的气味却越来越陌生。我一边闷头前行，一边害怕自己估错了方向。但我不敢回头，我怕一转头就贴上那户人家女主人的脸。

星星消失了，隔着树木，我看到路上的车子越来越多，声音越来越大。我爬上一棵树，回头看，那边的天空已经露出红色。我该停下来休息休息了，昨天傍晚在垃圾堆吃的剩饭已经消化并融在我稍稍肥硕的身体里。短短十多天里，我发了些福。

猫姑娘(五)

发表于 2021-04-26 分类于 小说
本文字数： 2.1k 阅读时长 ≈ 2 分钟

我蹲在路边的两棵冬青之间，眯着双眼思考。前方车流涌动。当它们要慢下来的时候，屁股会红起来。起初我很惧怕，后来明白它们无意伤害我，只要我遵循它们前进的规律。夜幕就要降临，西方的暮色快要燃尽了。我认得出西方有什么用，我不知道家的方向。

我也没想到我的演技能这么超群。那户人家真的以为我爱上了他们，否则不会这么信任我，对那扇敞开的门置之不理。在我逃离之前，那扇门曾无数次敞开紧接着关上，我用耳朵聆听门轴的旋转，目光却在那小女孩身上。我收起手掌下的尖刺，四脚朝天，眼斜嘴歪，把那女孩儿逗引的咯咯笑，然后用手掌轻轻拍打她的手臂，并发出娇嗔埋怨而顽皮的喵喵声。这都是我曾经玩烂了却未曾奏效的戏码，如今它们竟派上了用场。

我被“嗷”的一嗓子惊醒，顺势跳下窗户。

原来是主人醒了。他“当”的一声推开屋门，把我抱起来，左晃右晃，然后嘟嘟囔囔地不知说些什么。我从他惊喜的脸看出，把我送给别人不是他的主意。这倒是件让我开心的事情。

无论如何，我回来了。在这个清凉的早晨，我还史无前例地被主人奖励了一个完整的煎鸡蛋。在主人走去上学后，我找到了我的男朋友。他对我的离开一无所知，他说他一晚上都没找到吃的，到现在还饿得不行。看来，食物比我更重要。于是我陪他去南面那片葡萄园觅食。

猫姑娘(二)

发表于 2021-04-26 分类于 小说
本文字数： 1.8k 阅读时长 ≈ 2 分钟

跳入黑暗，发现世界很大。

我的这帮同类，我曾鄙视的这帮同类，他们都有各自的行军路线。傍晚他们蹲在垃圾堆旁边的墙上，等候人类送来晚餐。但那无法喂饱他们。到了深夜，他们兵分两路，一南一北。南面是一片葡萄园，葡萄园里有蚂蚱、老鼠……和青色的葡萄串；北面是密集的居民区，有更多垃圾堆，有更多更丰富的剩菜。我更喜欢南面。到了晚上，那是一个繁华的世界，花草的味道很好闻，心情好的时候我会吃上两口；奇妙的声音让我不能自持，除了蚂蚱和老鼠，我还曾跟随那声音找到过蟋蟀、小蛇和青蛙，不过，他们最后都被我玩死了。我还会蹲在葡萄藤上面凝视黑夜和星星。在同类的眼里我是个怪胎，我每天夜里跟着他们，却不干正经的事情：觅食。我像一个傻猫，整夜沾花惹草逗蛐蛐，却不懂得吞下该吞下的东西。

一片想飞的叶子

发表于 2021-04-25 分类于 诗
本文字数： 150 阅读时长 ≈ 1 分钟

一片叶子想要飞
它还浑身浅浅的绿色

我怎么才能飞呢？
我想飞上天看看我所在的这棵树，这片森林

你使劲儿长，长得大大的
在秋天就能飞的很高很远

叶子于是卯足了劲儿长
长得像一页纸这么大
在一个秋风瑟瑟的午后
叶子决定起飞
它挣脱了树
却被一阵莫名其妙的风吹入河里

河水带着它漂流
漂了大半个中国
最后它沉入水底

失落

最后一片叶子离开我
昨夜的繁星在昨夜闪着
风安静地走远
世界已沉默

我走遍所有的记忆
找不见落脚之地
像一个被限时的游客
匆忙而不舍

笑容都定格在僵硬的脸上
走吧走吧 他们只是雕塑
灵魂早已出了壳
